# Decisions index

## Accepted

- [Headless scientific identity is not GUI lifecycle state](002-headless-scientific-identity-is-not-gui-lifecycle.md)
  — reusable source requests use registered asset/content identity and remain
  importable without Qt; GUI lifecycle generations remain transient
  publication state outside the headless source contract.
- [Reusable benchmarks are product diagnostics](001-reusable-benchmarks-are-product-diagnostics.md)
  — reusable timing and capacity measurements belong behind package APIs and
  supported CLI or UI surfaces; scripts are reserved for disposable
  experiments and fixture construction.
